# BragForge 🛠️

**Forge your raw git logs and daily work into impactful, metric-driven brag sheets and performance review bullets.**

BragForge is a lightweight, responsive, and aesthetic Single-Page Application (SPA) designed to solve a universal developer pain point: career tracking. Instead of digging through months of Git history and Jira tickets at performance review time, paste your raw git log and let BragForge parse, structure, and format your impact using the STAR method.

## The Problem
Developers write code daily, but summarizing their value during self-assessments, performance reviews, or resume updates is an annoying, time-consuming chore.
- Commit logs are too granular and technical (e.g., `fix: socket connection timeout`).
- Non-technical stakeholders need to see high-level business impact (e.g., "Improved WebSocket connection stability by 40%, reducing user reconnection drops").
- Remembering accomplishments from 6 or 12 months ago is highly error-prone.

## The Solution: BragForge
BragForge makes it trivial to:
1. **Parse Git Logs**: Automatically categorize conventional commits (`feat`, `fix`, `refactor`, `perf`, etc.).
2. **Quantify Impact**: Prompt developers to add quantitative metrics and business value to their tasks.
3. **STAR Generator**: Formulate accomplishments into standard **STAR** (Situation, Task, Action, Result) bullets.
4. **Export Clean Layouts**: Generate formatted Markdown, JSON config, or styled HTML pages ready for distribution.

## Key Features
- **Sleek Dark Mode & Glassmorphic UI**: Beautiful, interactive interface with smooth transitions.
- **Git Commit Regex Parser**: Processes both raw commit lines and conventional commits.
- **Interactive Achievement Board**: Edit, prioritize, and category-tag your work.
- **Live Preview & Styling**: Instantly preview how your bullets will render in different styles.
